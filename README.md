# mwgg_igdb — sixteen variant (2026.05.01)

This branch is the published `sixteen` variant of the MultiworldGG game index.
It is force-pushed by the daily-release workflow in the Index repo and tagged
on each push as `sixteen-YYYY-MM-DD`.

Install with:

```
pip install git+ssh://git@github.com/lallaria/MultiworldGG-Index@game_index_sixteen
```

The package exports a `_GameIndexClass` singleton at `mwgg_igdb.GameIndex`.
