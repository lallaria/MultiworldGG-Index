# mwgg_igdb — twelve variant (2026.05.01)

This branch is the published `twelve` variant of the MultiworldGG game index.
It is force-pushed by the daily-release workflow in the Index repo and tagged
on each push as `twelve-YYYY-MM-DD`.

Install with:

```
pip install git+ssh://git@github.com/lallaria/MultiworldGG-Index@game_index_twelve
```

The package exports a `_GameIndexClass` singleton at `mwgg_igdb.GameIndex`.
